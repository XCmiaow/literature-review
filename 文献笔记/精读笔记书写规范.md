> **建立日期**: 2026-04-08

> **适用项目**: 宽温度容忍性纤维素基凝胶材料综述

---

## 一、规范目的

本规范旨在：

1. **统一笔记格式** — 确保三姐妹的笔记风格一致
2. **保证内容质量** — 每篇笔记都包含必要的分析维度
3. **便于协作维护** — 妹妹们可以独立完成笔记
4. **积累可复用素材** — 提取学术句式供综述写作使用

## 三、笔记结构规范

每篇精读笔记必须包含以下**七大模块**：

### 模块 1: 论文基本信息表

```markdown
| 项目 | 内容 |
|:-----|:-----|
| 标题 | [英文标题] |
| 作者 | [第一作者 et al.] |
| 期刊 | [期刊名, 年份, 卷, 页] |
| DOI | [DOI号] |
| 通讯作者 | [姓名+单位] |
| 精读日期 | [YYYY-MM-DD] |
| 精读人 | [桃桃Claw/千桃Claw/PeachClaw] |
```

### 模块 2: 一句话摘要

用**1-2句话**概括论文的核心贡献，不超过50字。

> 示例：本征抗冻水凝胶领域经典综述，系统总结了盐/醇/离子液体/聚合物修饰四大抗冻策略及其机理。

### 模块 3: 核心创新点评析

**必须回答三个问题**：

1. **这篇论文解决了什么问题？**（< 30字）
2. **核心创新点是什么？**（50-100字）
3. **为什么这个创新重要？**（30-50字）

### 模块 4: 关键技术路径

用**图表或流程图**说明论文的技术路线。

```
问题 → 方法 → 核心机理 → 关键结果
```

### 模块 5: 关键数据提取表

必须包含的指标：

| 必填指标 | 说明 |
|:---------|:-----|
| 温度范围 | 抗冻/耐温上/下限 |
| 力学强度 | 拉伸强度或模量 |
| 离子电导率 | 单位：mS/cm |
| 循环稳定性 | 次数+保持率 |

选填指标：根据论文实际内容补充

### 模块 6: 对综述手稿的贡献

**必须回答**：

1. 这篇文献能用在综述的哪个/哪些章节？
2. 用什么形式呈现？（表格对比、示意图、数据引用？）
3. 有哪些学术句式可以提取供综述写作用？

### 模块 7: 个人独立思考

**必须包含**：

1. **专家视角**：如果你是审稿人，会如何评价这篇论文？
2. **局限性**：这篇论文有哪些不足或未解决的问题？
3. **留给课题组的问题**：对课题组后续研究有什么启发？

## 四、内容质量标准

### 4.1 及格线（必须满足）

- [ ] 论文信息完整（标题、作者、期刊、DOI）
- [ ] 有核心创新点的描述
- [ ] 有关键数据提取
- [ ] 有对综述章节的贡献分析
- [ ] 有至少1条学术句式摘录

### 4.2 良好（建议满足）

- [ ] 有机理分析（不只是描述现象）
- [ ] 有与其他文献的对比
- [ ] 有个人独立思考
- [ ] 学术句式>=3条

### 4.3 优秀（加分项）

- [ ] 有深度图片解析
- [ ] 有对课题组工作的具体建议
- [ ] 有对未来研究方向的预判
- [ ] 主动关联手稿框架

## 五、写作风格规范

### 5.1 语言要求

- **中文为主**，关键术语可中英混排
- **学术语气**，避免口语化
- **客观描述**，不夸大贡献

### 5.2 禁用语

```
❌ "这篇论文很厉害"
❌ "作者做了很多实验"
❌ "效果非常好"
❌ "总之很棒"

✅ "该策略实现了...性能"
✅ "系统研究了...的影响"
✅ "表现出优异的...特性"
```

### 5.3 学术句式摘录规范

摘录的句式必须：

1. 是论文原文的**准确翻译或改写**
2. 描述的是**通用性机理或方法**
3. 标注来源（可用斜体引用）

## 七、常见问题

### Q1: 论文PDF太大打不开怎么办？

A: 使用`pdftotext`命令提取文字：
```bash
pdftotext -layout paper.pdf - | head -200
```

### Q2: 遇到不懂的专业术语怎么办？

A: 先查中文综述/教材建立概念，再用英文文献深化理解

### Q3: 精读一篇论文需要多久？

A: 深度精读约2-3小时，泛读约30分钟
